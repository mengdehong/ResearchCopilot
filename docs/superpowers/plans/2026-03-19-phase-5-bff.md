# Phase 5: FastAPI BFF Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development or superpowers:executing-plans.

**Goal:** 实现 FastAPI BFF 层：认证中间件、Repository 层、API Router、SSE 事件流、文件代理。

**Architecture:** 分层架构 `Router → Service/Repository → DB`。FastAPI 的 `Depends()` 注入 Session 和服务实例。
